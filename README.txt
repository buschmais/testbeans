TestBeans aims to integrate JUnit 4.9 and later with Contexts and Dependency Injection (JSR-299)
using the reference implementation Weld. It provides three new annotations which allow binding
beans (e.g. resources) to test-related contexts: @SuiteScoped, @ClassScoped and @MethodScoped.

For further information refer to https://github.com/buschmais/testbeans/wiki/TestBeans.
